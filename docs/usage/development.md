### Start API & Prisma server

1. Open a terminal
2. Start Prisma with docker-compose
3. Start API server

```bash
cd PROJECT_NAME
cd api && docker-compose up-d && npm run dev
```

### Start Website server

1. Open a new terminal
2. Start website server

```bash
cd PROJECT_NAME
cd website && npm run dev
```
